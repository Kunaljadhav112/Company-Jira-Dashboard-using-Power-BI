# Company-Jira-Dashboard-using-Power-BI
Built a real-time Jira Dashboard in Power BI by integrating data directly from the Jira REST API, enabling automated reporting, campaign tracking and performance analysis for improved project visibility.

I am sharing a version that was previously used for testing and demonstration purposes (data is not updated). The production dashboard contains real-time and confidential company data, which is directly pulled into Power BI via the Jira REST API. As per company policy and data confidentiality guidelines, I am not authorized to share the fully updated or live production version.
